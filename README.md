# OumyfRecettes

Monorepo de l'application OumyfRecettes.

## Structure

- `src/`, `public/` : application web Vite/React
- `api/` : backend Strapi
- `mobile/` : application mobile Expo / React Native

## Démarrage

### Web

1. `npm install`
2. `npm run dev`

### API

1. `cd api`
2. `npm install`
3. `npm run develop`

### Mobile

1. `cd mobile`
2. `npm install`
3. `npx expo start`
